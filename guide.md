# Simple Guide

## Tmux Controls

**Main key**: Ctrl-a (press first, then the other key)

### Splits
- **Ctrl-a |** → split vertical (side by side)
- **Ctrl-a %** → split horizontal (top and bottom)

### Moving Around
- **Ctrl-a ←** → go left
- **Ctrl-a →** → go right  
- **Ctrl-a ↑** → go up
- **Ctrl-a ↓** → go down

### Make Panes Bigger/Smaller
- **Ctrl-a w** → make taller
- **Ctrl-a s** → make shorter
- **Ctrl-a a** → make wider left
- **Ctrl-a d** → make wider right

### Zoom
- **Ctrl-a +** → make pane full screen
- **Ctrl-a -** → make pane normal size

### Other Stuff
- **Ctrl-a r** → reload settings
- **Ctrl-a c** → new window
- **Ctrl-a x** → close pane

## Vim Controls

### File Tree
- **\nt** → show/hide file tree

### Search Files
- **\ff** → search in all files

### Split Screen
- **:vs** → split vertical
- **:hs** → split horizontal
